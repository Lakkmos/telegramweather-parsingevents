Запустить с библиотеками из requirements
Бот принимает дату кириллицей, без года. Например, "3 августа" и отвечает информацией о погоде на указанную дату. Информация берется с Яндекс.Погода
Так же отвечает на команду "/events", формируя список статей из vk.com/@yvkurse в файл events.xls, находящийся рядом с weather.py и высылая список в чате.
